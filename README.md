# 人生重开 · 选专业版 🎲

> 报志愿之前，先替未来的自己，把这个专业的人生**活一遍**。

一款给高考生的 AI 文字游戏。用 10–15 分钟"活过"某个专业从大一到 35 岁的一生，看清**同一个专业的人，命运会怎样分化**，最后带走一份"这个专业适不适合你"的诚实反馈。

**v1 只做「计算机」专业。**

---

## 这是什么

绝大多数高考生在报志愿时，对"某个专业毕业后到底过什么样的生活"几乎一无所知——全靠道听途说、短视频和刻板印象。这个游戏想填补那个信息鸿沟：

- 🎮 **纯选项式**：开局填几个真实的"出身设定"（学校档次、家境、城市、是否有行业人脉、抗压能力），然后一路做选择，活过三幕人生。
- 🔀 **命运会分化**：同一个专业，出身不同、选择不同，结局天差地别——有人进大厂、有人考公上岸、有人在外包里被时代追着跑。游戏让你**提前看见这种分化**。
- 🪞 **结局给三层反馈**：照见你自己（用你的选择回放）→ 退一步的理性提醒（风险点 + 你该提前想清楚的问题）→ 三句过来人会跟你说的不同的真话。

### 设计原则（写进了游戏的"铁律"里）

- **给地图，不给判决书**：出身差的牌面也有改命的关键节点和尊严，绝不做成"出身决定论"。
- **不贩卖焦虑**：35 岁不做成倒计时恐怖装置，AI 冲击不渲染成"必然失业"，"普通而有尊严的结局"要演饱满。
- **不打匹配度分数**，不下命运预言——只帮你把该想的问题想清楚。

---

## ⚠️ 原型阶段声明

当前是**早期原型**，内容真实性靠大模型的行业知识 + 游戏提示词里的设计铁律兜底，**尚未经过查证内容包核实**，也未经行业从业者审校。

**它能帮你提前想几个问题，但不能预测你的人生，更不构成任何报考建议。** 请务必结合真实的老师、家长、从业者的意见来做决定。

## 怎么玩

核心是一段自包含的「游戏提示词」（见 [`prompts/game-prompt-zh.md`](prompts/game-prompt-zh.md)），可以喂给任意中文大模型。三种玩法：

### DeepSeek（最简单）

打开 DeepSeek 对话，把 [`prompts/game-prompt-zh.md`](prompts/game-prompt-zh.md) 里 `===` 之间的全部内容粘贴进去发送，它就会开始问你出身、带你玩。

### 豆包 / 扣子（做成可分享的智能体）

用字节的 [扣子 Coze](https://www.coze.cn/) 把它做成一个能发布到豆包、可分享链接的智能体：

1. 登录扣子 → **创建智能体**。
2. 把上面 `===` 之间的内容粘进 **「人设与回复逻辑」**。
3. **模型**选豆包或 DeepSeek；**开场白**填那句"这是一个 10 分钟的简化模拟……"。
4. （可选）把"计算机内容骨架"那段存成文档传进 **知识库**，让内容和指令解耦、好维护。
5. **发布** → 选「豆包」渠道，拿到分享链接发出去即可。

### Claude Code

仓库里也带了一个 Claude Code skill（`.claude/skills/zhuanye-rensheng-tiyan/`）。把它复制到 `~/.claude/skills/`，在 Claude Code 里输入 `/zhuanye-rensheng-tiyan` 即可游玩。
