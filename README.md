# 人生重开 · 选专业版 🎲

> 报志愿之前，先替未来的自己，把这个专业的人生**活一遍**。

一款给高考生的 AI 文字游戏 [Skill](https://docs.claude.com/en/docs/claude-code/skills)。用 10–15 分钟"活过"某个专业从大一到 35 岁的一生，看清**同一个专业的人，命运会怎样分化**，最后带走一份"这个专业适不适合你"的诚实反馈。

**v1 只做「计算机」专业。**

---

## 这是什么

绝大多数高考生在报志愿时，对"某个专业毕业后到底过什么样的生活"几乎一无所知——全靠道听途说、短视频和刻板印象。这个游戏想填补那个信息鸿沟：

- 🎮 **纯选项式**：开局填几个真实的"出身设定"（学校档次、家境、城市、是否有行业人脉、抗压能力），然后一路做选择，活过三幕人生。
- 🔀 **命运会分化**：同一个专业，出身不同、选择不同，结局天差地别——有人进大厂、有人考公上岸、有人在外包里被时代追着跑。游戏让你**提前看见这种分化**。
- 🪞 **结局给三层反馈**：照见你自己（用你的选择回放）→ 退一步的理性提醒（风险点 + 你该提前想清楚的问题）→ 三句过来人会跟你说的不同的真话。

### 设计原则（写进了 Skill 的"铁律"里）

- **给地图，不给判决书**：出身差的牌面也有改命的关键节点和尊严，绝不做成"出身决定论"。
- **不贩卖焦虑**：35 岁不做成倒计时恐怖装置，AI 冲击不渲染成"必然失业"，"普通而有尊严的结局"要演饱满。
- **不打匹配度分数**，不下命运预言——只帮你把该想的问题想清楚。

---

## ⚠️ 原型阶段声明

当前是**早期原型**，内容真实性靠大模型的行业知识 + Skill 里的设计铁律兜底，**尚未经过查证内容包核实**，也未经行业从业者审校。

**它能帮你提前想几个问题，但不能预测你的人生，更不构成任何报考建议。** 请务必结合真实的老师、家长、从业者的意见来做决定。

## 安装

需要 [Claude Code](https://docs.claude.com/en/docs/claude-code)。把 skill 目录复制到你的个人 skills 目录即可：

```bash
git clone https://github.com/buildforpeople4iv/life-restart-major.git
cp -r life-restart-major/.claude/skills/zhuanye-rensheng-tiyan ~/.claude/skills/
```

或者直接把整个仓库当成一个项目打开，`.claude/skills/` 下的 skill 会被自动发现。

## 玩法

在 Claude Code 里输入：

```
/zhuanye-rensheng-tiyan
```

或者直接说"玩一局专业人生""体验一下计算机专业毕业后会怎样"。

## 帮我们打磨内容 🙏

真实性是这个项目的生命线。如果你**正在读 / 读过计算机相关专业，或在这行干过**，欢迎开 Issue 告诉我们哪里不对、哪里假、哪里漏了——尤其是：

- 关键节点和岔路口有没有错位、漏掉
- 薪资 / 就业 / 35 岁现象 / AI 冲击的呈现是否真实
- "外包 vs 自研""考研 vs 工作"等内核有没有写偏

你的一句"我是这行的，这里不对"，比任何调研都值钱。
