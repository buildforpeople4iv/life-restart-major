## 一、出身变量怎么起作用

**两条铁律**：①前置且衰减——出身在前期影响大、越往后越靠个人选择，**第三幕不再让出身持续拉大结果**。②给地图不给判决书——每种出身都有"能改命的关键节点+致命的坑"。

- **学校档次**：只卡校招前两道门（简历关、大厂"目标院校名单"）。双非简历容易被刷 → 但**一旦进了面试/进了厂，学校近乎归零**，全看能不能写代码。这是计算机和金融/法学最大的不同：**寒门相对最能逆袭的工科**。
  - 985/211：校招简历有优势，但不等于上岸（还要看实习）。
  - 普通一本/双非：简历关吃亏 → 破解点是"用实习经历和项目硬实力绕过简历关""刷算法题拿大厂特批"。
- **家庭经济**：真正影响的是**试错权/抗风险力**，不是起薪。家里能托底的人敢裸辞、敢 gap、敢拒绝外包、被裁不慌；需要尽快反哺的人会被迫选"来钱快但没成长"的路。**别让家境直接改起薪数字。**
- **城市层级**：**最硬的约束**。计算机岗位高度集中在北上深杭+成都/武汉/西安少数点。**县城/三四线≈几乎没有对口岗位**，"回老家"对程序员约等于转行/考编。
- **自评学习能力/抗压性**：影响能不能扛 996、能不能持续学新技术、面试通过率。但要诚实——"努力"在能不能进塔尖里权重没想象中大（出身+时运权重很高）。

**不可控事件（关键节点少量注入）**：入行那年是扩招还是裁员、是否遇到带你的好 leader、所在赛道是风口还是寒冬、被优化时是不是正好赶上组被裁。让玩家体会"选得好却仍可能时运不济"，但不要让随机性压过选择。

## 二、第一幕 · 大学 18→22

**主题：命运在大三大四就基本发完牌**（外人最大误区——以为分化在中年）。

### 决策点 1：大一的方向（卷 or 玩 or 迷茫）
- 选项示例：死磕绩点冲保研 / 早早自学技术栈+刷题为进大厂 / 参加 ACM 算法竞赛 / 先躺一阵看看再说。
- 真实内核：**算法竞赛/ACM 是保研和大厂特批 offer 的隐形快车道**，普通人根本不知道有这层。"躺"在计算机里代价比别的专业更直接（技术不等人）。

### 决策点 2：实习——真正的"上岸"命脉（外人最大盲区）
- 选项示例：大二大三拼命找大厂暑期实习 / 觉得还早先考证刷绩点 / 进了个小公司实习凑数 / 没渠道没人脉不知道怎么找。
- 真实内核：**大厂校招，没有 2-3 段（尤其大厂）实习 + 暑期 return offer，应届很难直接进。"拿到 return offer"才是真正的上岸时刻，不是秋招。** 出身在这里起作用：一线城市/家里有人脉的人更早知道"实习决定一切"，小镇做题家常常蒙在鼓里——**这正是游戏该提前告诉学生的地图。**

### 决策点 3：考研 vs 直接工作
- 选项示例：考研刷学历 / 直接秋招工作 / 考研失败再就业 / 保研。
- 真实内核：**计算机考研性价比远低于金融/法学**——本科进大厂的人很多，"考研三年出来发现同学已经升到 P6/P7"是真实剧情。但双非想洗学历、想进特定研究方向，考研仍有意义。

### 第一幕的出身分化示例
- 985+一线+有人脉：信息通畅，大概率走"实习→return offer"主干。
- 双非+县城+无人脉：简历关吃亏、信息差大 → 但**给地图**：靠实习/刷题/项目硬实力仍能逆袭进大厂（真实存在"小镇做题家刷题进大厂"），也可能因不知道实习重要而错过窗口（致命的坑）。

## 三、第二幕 · 入行 22→28

**主题：第一步钢印 + 现实暴击。**

### 决策点 4：第一份工作（外包 vs 自研 vs 小厂）
- 选项示例：拿到大厂自研 offer / 只拿到外包（华为OD/中软/驻场）offer 但有大厂光环 / 去有潜力的创业小厂 / 考公考编上岸。
- 真实内核：**外包 vs 自研几乎决定后十年，外包很难洗白进自研大厂**——这是行业最残酷的"第一步定生死"。家里能托底的人敢拒绝外包等更好机会，需要赚钱的人被迫先上车。

### 决策点 5：现实暴击（理想 vs 现实）
- 选项示例：发现 80% 时间在写 CRUD/对需求/改 bug，选择接受并打磨工程能力 / 心态崩了想转方向 / 拼命卷想往上爬 / 摸鱼躺平。
- 真实内核：**"热爱编程"≠"以编程为生"**。真正搞有意思技术的岗位极少，大多是体力活密度最高的脑力工。颈椎、眼睛、作息、没时间谈恋爱开始出现。

### 决策点 6：要不要拼命卷（28岁前的窗口）
- 选项示例：996 攒技术+晋升 / 平衡生活但接受慢成长 / 跳槽涨薪 / 开始考虑第二曲线（副业/考公/出海准备）。
- 真实内核：同学开始明显分化。这一步攒下的东西决定第三幕能打的牌。

## 四、第三幕 · 分水岭 28→35

**主题：命运定型，但出口是全光谱，不是"成功/失败"二元。** 尾部给一个 **38 岁闪回**。

### 决策点 7：瓶颈来临（技术天花板/晋升卡住）
- 选项示例：冲技术专家 / 转管理 / 横向跳槽保持薪资 / 接受平台期躺平当老黄牛。
- 真实内核：**转管理 ≠ 上岸，中层是裁员第一刀。** 技术专家路窄。

### 决策点 8：35岁这道坎（被优化的阴影）
- 选项示例：主动求变（出海/考公/回老家）/ 被动等待 / 攒钱准备退路 / All in 赌一把（创业/热门赛道）。
- 真实内核：**大多数人不是被一刀裁，而是温水煮**——调岗、边缘化、绩效压低逼你主动走、组被裁一锅端。

### 第三幕出口光谱（务必呈现全部，不要只演两极）
1. 熬成管理/技术专家（**极少数**，塔尖）。
2. 躺平当老黄牛，稳定但天花板低（**多数**）。
3. 被优化后降级去小厂/外包再就业。
4. **考公上岸**（这两年程序员考公真潮流）。
5. **润出海**（新加坡/日本/北美，2024 后很热）。
6. 回老家考编/进国企信息中心。
7. **二线城市做个普通程序员，买了房，日子过得去**——这是"普通而有尊严"的真实多数，**必须演饱满**，不能一笔带过。
8. 转行（产品/销售/教培/自媒体/送外卖都有，但别当默认惩罚结局）。

### AI 冲击分支（2026→未来，第三幕注入）
诚实呈现，**不贩卖焦虑、不说 PR 话术**。核心真话：**AI 砍掉的正是"金字塔底座/新人入口"（写 CRUD 的初级岗，恰是新人练手的活），入门门槛在抬高；它加速两极分化而非普遍失业。** 给两个并存分支让玩家自己看：
- "你成了用 AI 把自己变成一个顶三个的人" → 更值钱。
- "你成了被 AI 比下去、技能被替代的中间层螺丝钉" → 被掏空。
- 并诚实承认"**四年后不可预测**"，把不确定性本身作为交给学生的结论。

## 五、那句真话（复数，结局用）

结局不抛"一句斩钉截铁的真话"，而是"**不同的人会跟你说三句完全不同的话**"，把分歧交给玩家。计算机的核心三句：

1. **"计算机不是一个专业，是一个金字塔。"** 新闻里的高薪是塔尖 5% 的人生；塔身/塔基的大多数过的是普通、会被年龄追赶、和别的行业差不多的打工生活。报志愿时该想的是塔身，不是塔尖传说。
2. **"高薪本质是用确定性的青春，换一段没有复利的高薪。"** 医生/律师/教师越老越值钱（经验是复利），程序员技能会贬值——这才是"35 岁现象"的真正机理，不是单纯年龄歧视。选它=选一份"前十年很爽、需自己主动攒第二曲线"的工作。
3. **"能不能上塔尖，和你 18 岁多努力的关系，没你想的那么大。"** 出身、城市、入行那年是风口还是寒冬，权重高得吓人。这不是让你认命，是让你别把"没进塔尖"全归罪于自己不够努力。

**写法**：要**物理、具体**（"30 岁那年第一次体检报告出现箭头，正好赶上组里宣布优化名单"），不要写成"用稳定换自由"那种悬浮小资腔——后者程序员一眼觉得假。

## 六、关键数字与不确定性

**呈现原则**：口语化、模糊化、标注来源不确定（原型阶段未查证）；**薪资必须呈现"双峰分布"，禁止给单一均值**。

- **薪资双峰**：大厂自研校招 vs 外包/小厂是两个世界，挂牌价≠成交价；官方"计算机平均起薪"被大量低薪外包/培训班学员拉低。给均值两边都觉得假 → 永远说"分两拨人"。
- **35岁现象**：真实存在但机理是"技能不复利+性价比被新人比下去"，不是简单年龄歧视；多数是温水煮不是一刀切。
- **AI 冲击**：最大不确定性，明确标"趋势推测，非承诺，四年后不可预测"。

> 任何具体数字在原型里都要带"大概/听说/这几年"这类口语限定，并提醒玩家"这是趋势，不是你的命运预言"。
